## 二、变量与简单数据类型

### 2.1 变量
1. 每个变量都会指向一个**值**，在程序中可以随时更改这个**值**，而python会始终记录变量的最新值。
2. 变量的命名规则与规范
  - 变量名只能用**字母、数字、下划线**。
  - 变量名不能用空格，得用下划线分隔单词。
  - 不能将**关键字**或者**函数名**作为变量名。
  - 变量名应既简短又具有描述性。
  - 慎用l和O，因为很容易被人看错。

### 2.2 简单数据类型

#### 2.2.1 字符串
1. 字符串就是一系列字符，python中用引号括起来的都是字符串。
``` python
"这是一个字符串"
'这也是一个字符串'
title() # 将所有单词的首字母大写
upper() # 将所有单词都大写
lower() # 将所有单词都小写

message = "hello"
newMsg = f"{message} world" # f帮助在字符串里使用变量
```
